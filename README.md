## 概要
ダニエル・バックリー氏の Block Breaker Template を基に、C# を使用して独自のブロック崩しゲームを制作しました。

## 追加・改良した主なポイント
- ボールとパドルの速度調整、3回まで再スタート可能なライフ制を追加
- ブロックに耐久値を導入し、色分けで状態が分かりやすいように改善
- 汚染区域・清潔区域を設置し、パドルの状態が変化するギミックを追加
- パドルの状態に応じて **汚染ボール / 清潔ボール** を打ち返せるように実装
- 汚染ボールでブロックを破壊すると、周囲に **新しいブロックが増殖**する感染システムを追加
- BGMに呼吸音を追加、ブロック数は回数に応じて速度が変化する機能を追加
